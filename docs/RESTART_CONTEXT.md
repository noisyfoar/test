# Контекст проекта (для перезапуска)

Этот файл нужен, чтобы быстро вспомнить состояние работ и без проблем продолжить после рестарта агента.

## Что это за проект

Мы делаем новую реализацию `dlisio` на C#.

- Основная цель: чтение DLIS (RP66).
- Целевая платформа: `.NET Framework 4.8`.
- Дополнительно поддерживаем сборку под `net8.0` для удобного запуска тестов в Cloud.

## Что уже сделано (простыми словами)

1. Репозиторий очищен от старого Go-кода.
2. Создан .NET-каркас:
   - `DlisioNet.sln`
   - `src/Dlisio.Core`
   - `tests/Dlisio.Tests`
3. Реализован базовый парсинг транспортного уровня DLIS:
   - LRSH (заголовок сегмента),
   - LRST (trailer сегмента: padding/checksum/trailing length),
   - полный парсинг одного сегмента (header + encryption packet + body + trailer),
   - чтение сегмента из stream.
4. Реализована сборка логической записи из нескольких сегментов:
   - склейка multi-segment цепочки,
   - проверка корректности цепочки (first/last, тип, флаги структуры/шифрования),
   - чтение следующей логической записи из stream.
5. Добавлена настройка cloud-окружения:
   - `.cursor/environment.json`
   - `.cursor/install-dotnet8.sh` (идемпотентная установка .NET 8)
6. Добавлены инструкции для агентов:
   - `AGENTS.md`, включая правило: после каждого шага запускать тесты и делать короткую рефлексию.
