# DLISIO.NET (план запуска)

Этот репозиторий очищен и подготовлен для новой реализации библиотеки
[equinor/dlisio](https://github.com/equinor/dlisio) на C#.

## Цели

- Реализовать чтение и декодирование DLIS (RP66) на C#.
- Целевая платформа: **.NET Framework 4.8**.
- Допускаются C++ вставки (например, C++/CLI или нативная DLL с P/Invoke)
  для производительных или сложных низкоуровневых частей.

## Текущее состояние

- Удалены старые Go-файлы из репозитория.
- Добавлен детальный план реализации в `docs/implementation-plan.md`.

## Следующий шаг

Согласовать архитектурный подход к interop:

1. `Чистый C#` (максимальная переносимость, ниже сложность сборки),
2. `C# + C++/CLI` (удобный мост к C++ коду, но привязка к Windows/MSBuild),
3. `C# + native C++ + P/Invoke` (гибко и быстро, но больше кода marshalling).
